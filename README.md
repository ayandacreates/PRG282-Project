# 🦸Superhero Database System
A C# Windows Forms application built for the One Kick Heroes Academy. The system replaces a paper-based process with a digital solution for managing trainee superhero records, tracking performance, and generating reports.

## 🔨Technologies 
C#, Windows Forms, Git, GitHub

## Features 
**Add a Hero**: Log new trainee superheroes with their relevant details

**View Heroes**: Browse and view all existing superhero records in the system

**Update a Hero**: Edit and update a hero's information when needed

**Delete a Hero**: Remove a hero's record from the system

**Rank & Threat Level Calculation**: The system automatically calculates each hero's rank and threat level based on their exam scores

**Summary Report**: Generates a report showing the total number of heroes, the average age and exam score, and how many heroes fall under each rank

## The Process
This was a group project built by a team of 4. The goal was to build a digital replacement for a manual, paper-based system used to record and assess trainee superheroes. We divided the features amongst ourselves, collaborated using Git and GitHub to manage changes and make meaningful commits, and brought everything together into one working application.

## 📚What I Learned 
This project taught me how to build a fully functional Windows Forms application using C#. I gained a better understanding of how to structure an application that handles real data — adding, viewing, updating, and deleting records. I also learned how to implement logic that automatically calculates values (ranks and threat levels) based on user input, and how to generate summary reports from that data. Working with Git and GitHub as a team also strengthened my understanding of version control and collaboration.

## What Can Be Improved
A few areas that could generally be improved in a project like this include replacing text file storage with a proper database so data is more secure and easier to manage, improving the UI to feel more modern beyond the default Windows Forms styling, and adding input validation to prevent incorrect data from being entered. A search and filter feature would also make it easier to find specific heroes as the list grows.
